## 🌌 About Me

I am an **AI/ML Engineer** and **Full Stack Developer** with a passion for designing, constructing, and scaling intelligent applications. My core expertise is in building **Agentic AI systems**, optimized **Retrieval-Augmented Generation (RAG)** pipelines, and production-grade deep learning architectures. By bridging the gap between state-of-the-art machine learning models and high-performance full-stack web applications, I deliver software that is both highly intelligent and exceptionally scalable.

- 🧠 **GenAI & Orchestration**: Designing autonomous agent topologies and highly accurate semantic search systems.
- 💻 **Modern Web Architectures**: Developing clean API layers with FastAPI/Node.js and component-driven frontends with React and TypeScript.
- ⚙️ **Production Engineering**: Implementing clean MLOps pipelines using Docker, AWS, and GitOps workflows via GitHub Actions.
- 👁️ **Computer Vision & Video Analytics**: Training and deploying customized object tracking models using YOLOv8 and OpenCV.
