## 🏹 About Me

> *"कर्मण्येवाधिकारस्ते मा फलेषु कदाचन (Focus on your duty/code, not just the fruits of your labor.)"*

I am an **AI/ML Engineer** and **Full Stack Developer** dedicating my work to engineering production-grade intelligent systems. By aligning state-of-the-art machine learning models with robust full-stack web architectures, I build applications that are highly autonomous, performant, and resilient.

- 🧠 **GenAI & Orchestration**: Architecting multi-agent systems and highly optimized RAG pipelines.
- 💻 **Modern Web Architectures**: Developing clean APIs using FastAPI/Node.js and interfaces with React.
- ⚙️ **GitOps & Deployment**: Constructing continuous integration and cloud deployments using Docker and AWS.
- 👁️ **Computer Vision**: Training and deploying customized object tracking models using YOLOv8 and OpenCV.
